# Config schema

The JSON Schema for `dockprox.yaml` lives at
[`dockprox.schema.json`](https://github.com/foomo/dockprox/blob/main/dockprox.schema.json)
in the repository root.

Add this line to your YAML for IDE autocomplete + validation:

```yaml
# yaml-language-server: $schema=https://raw.githubusercontent.com/foomo/dockprox/main/dockprox.schema.json
```
